## 1. 项目概述

微信小程序，基于天气和用户个人衣橱，通过 AI 智能推荐每日穿搭方案。

核心功能：
1. 上传人身照，AI 分析体型风格
2. 拍照上传衣服，抠图去背景后存入衣柜
3. 结合天气 + 衣柜，AI 推荐搭配方案 + 生成出装图
4. 通知提醒用户查看搭配
5. 用户采纳后存入衣橱，形成穿搭档案

## 3. 技术决策

| 决策项 | 选型 | 说明 |
|--------|------|------|
| 前端 | 微信小程序原生 | WXML + WXSS + JS |
| 业务后端 | 微信云开发 | 云函数 + 云数据库 + 云存储 |
| AI 推理 | MiMo-v2.5 API | OpenAI Chat Completions 兼容格式 |
| 衣服抠图 | rembg (U2-Net) / remove.bg API | CPU 可跑，或用免费 API |
| 搭配出装图 | Pillow 图片拼接 | 零成本，< 1 秒 |
| 天气数据 | 和风天气 API v7 | 免费版 1000 次/天 |
| 管理员配置 | 小程序内配置页 | 仅管理员手机号可见 |

### MiMo-v2.5 API 确认信息
- 地址：https://api.xiaomimimo.com/v1/chat/completions
- 认证：api-key header 或 Authorization: Bearer token
- 模型：mimo-v2.5（全模态理解：文本+图片）
- 结构化输出：response_format: {"type": "json_object"}
- 超参数：temperature [0, 1.5]，top_p [0.01, 1.0]
- 上下文窗口：1M tokens，最大输出 128K tokens
- 速率限制：RPM 100，TPM 10M
