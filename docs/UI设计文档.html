7. 数据流与状态管理

衣服状态机

pending -> uploading -> pending -> recognizing -> recognized / failed

识别失败可重试; recognized后用户可编辑分类/颜色

搭配方案状态机

generating -> pending -> 用户操作

accepted: 展示在已采纳tab
discarded: 不展示

搭配计数规则

match_count = outfits WHERE status=accepted AND items包含该衣服ID 的数量

上限: 2套; 显示: 角标"⭐已搭配N套"(N>0时)

一键搭配按钮

pendingCount===0 -> "一键搭配"

pendingCount>0 -> "再来一批"

generating -> spinner + "生成中"，disabled
